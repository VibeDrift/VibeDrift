# Brand assets

The official VibeDrift mark: a chevron, for the forward motion of shipping code.

| File | Use |
| --- | --- |
| `vibedrift-logo.svg` / `vibedrift-logo.png` | The mark on its yellow tile (`#f0c000`). Default, for light backgrounds. |
| `vibedrift-logo-dark.svg` | Dark variant (yellow chevron on `#111116`), for dark backgrounds. |
| `vibedrift-icon-512.png` | 512×512 raster icon. |

Brand color: **`#f0c000`**. Please keep the chevron proportions and don't recolor or distort the mark.
